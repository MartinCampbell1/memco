# Private Single-User Contract

Status: current repo-local contract.

This contract defines the accepted current target for Memco in this checkout.

In scope:

- one private technical owner/operator;
- local runtime and repo-local configuration;
- seed-once/import flows for personal memory sources;
- review-gated candidate approval and safe publish;
- retrieval-only agent memory context through API/Hermes integration;
- evidence-backed facts, lifecycle operations, rollback, backup checks, and local audit artifacts.

Out of scope for the current private contract:

- public SaaS operation;
- user registration, billing, team administration, and broad multi-tenant product UX;
- fully autonomous memory mutation without review gates;
- claiming strict original/PDF parity;
- using fixture/mock evaluation as release-grade live proof.

Release claims must be based on fresh local verification for the current checkout and config, not on historical dated GO documents.
