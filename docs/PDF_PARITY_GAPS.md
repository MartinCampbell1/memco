# PDF Parity Gaps

Status: current reference-track gap list.

Memco is not strict PDF/original-brief complete. The current accepted target is the private single-user contract, not broad platform parity.

Known reference-track gaps:

- Docker Compose remains a strict original-brief mismatch for environments that require Docker-based deployment.
- Multi-user SaaS features such as registration, billing, team administration, and password login are intentionally out of current scope.
- The LoCoMo-like evaluation is an internal gate, not a paper-equivalent benchmark claim.
- Psychometrics support remains conservative and evidence-gated; it is not a full diagnostic or full PDF parity implementation.
- Public production hardening, hosted operations, and broad administrative UX remain outside the current private-local contract.

Use this file to separate acceptable private single-user simplifications from unfinished strict-reference work.
